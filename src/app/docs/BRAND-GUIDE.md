# BRAND GUIDE

**Project:** Heroes or Henchmens

**Organization:** The Order

**Version:** 1.0.0

**Status:** FOUNDATION FROZEN

---

# 1. Purpose

The Brand Guide defines the visual identity of The Order.

It establishes the visual language shared by every interface, artifact, document, glyph and communication produced inside the Heroes or Henchmens universe.

This document complements the Order Design System.

It never replaces it.

---

# 2. Brand Philosophy

The Order is not a corporation.

It is not a military organization.

It is not a religion.

It is an ancient institution dedicated to preserving, classifying and protecting knowledge.

Every visual decision must reinforce these principles.

---

## Authority

The Order speaks with certainty.

Never aggressively.

Never emotionally.

Always objectively.

---

## Precision

Everything exists for a reason.

Nothing is ornamental without meaning.

---

## Mystery

Information is revealed gradually.

Curiosity is encouraged.

Confusion is never intentional.

---

## Permanence

The Order existed long before the user.

It will continue to exist long after.

Its visual identity must feel timeless.

---

# 3. Brand Personality

The Order is:

• Intelligent

• Ancient

• Methodical

• Silent

• Precise

• Neutral

• Protective

Never:

• Funny

• Loud

• Chaotic

• Corporate

• Minimalistic for fashion

• Futuristic for spectacle

---

# 4. Visual Keywords

The visual identity is defined by:

Institution

Archive

Geometry

Precision

Balance

Hierarchy

Knowledge

Preservation

Observation

Memory

---

# 5. Design Language

Every interface should communicate:

Order

↓

Knowledge

↓

Documentation

↓

Discovery

↓

Power

Never the opposite.

---

# 6. Color Philosophy

Colors communicate state.

Never decoration.

Each color possesses semantic meaning.

---

## Primary

Represents The Order.

Used in:

Frames

Titles

Glyphs

Official Elements

---

## Secondary

Supports hierarchy.

Used in:

Subtitles

Secondary Information

Supporting Graphics

---

## Accent

Highlights important information.

Never dominates the interface.

---

## Status Colors

Status colors communicate document state.

Examples:

Verified

Restricted

Corrupted

Archived

Warning

Unknown

---

# 7. Typography

Typography communicates authority.

Primary Typeface

Institutional

Readable

Stable

Secondary Typeface

Technical

Monospaced

Metadata

Codes

Never use decorative typography.

---

# 8. Iconography

Icons are symbolic.

Not illustrative.

Icons must follow:

Simple geometry

Consistent stroke

CurrentColor

No gradients

No shadows

No realism

---

# 9. Glyph Language

Glyphs are not logos.

Glyphs are conceptual symbols.

Each Glyph represents an idea.

Examples:

Origin

Balance

Containment

Observation

Connection

Memory

Energy

Transformation

Glyphs never represent decoration.

---

# 10. Document Identity

Every Codex document belongs to The Order.

Never to the archived object.

The document identity remains constant.

Only its content changes.

---

# 11. Artifact Identity

Artifacts possess their own identity.

The document presents them.

It never imitates them.

---

# 12. Motion Language

Movement is subtle.

Animations should communicate:

Activation

Scanning

Discovery

Confirmation

Containment

Never entertainment.

Animation must never distract from information.

---

# 13. Sound Philosophy

If sound exists:

It must resemble instrumentation.

Mechanical systems.

Ancient technology.

Signal transmission.

Never arcade.

Never fantasy magic.

Never science-fiction clichés.

---

# 14. Imagery

Images should appear:

Archived

Catalogued

Observed

Documented

Not photographed.

Not cinematic.

Not promotional.

---

# 15. Symbol Hierarchy

Visual importance:

The Order

↓

Codex

↓

Artifact

↓

Glyph

↓

Decorative Assets

The institution always comes first.

---

# 16. Logo Usage

The Order Symbol represents the institution.

The Codex represents documentation.

Artifacts never replace institutional identity.

---

# 17. Visual Consistency

Every screen must be recognizable as belonging to The Order.

Without reading a single word.

Consistency is more important than novelty.

---

# 18. Brand Voice

Text should be:

Objective

Formal

Concise

Technical

Observational

Never emotional.

Never exaggerated.

Never promotional.

---

# 19. User Experience

The user should feel:

Curious.

Safe.

Guided.

Respected.

Never overwhelmed.

Never manipulated.

---

# 20. Golden Rule

Every visual decision must answer:

"Does this strengthen the identity of The Order?"

If the answer is no,

the decision should be reconsidered.

---

END OF DOCUMENT.
